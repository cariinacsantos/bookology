## 🧰 Tech Stack

| Frontend  | Backend        | UI & Styling      | Other Features           |
|-----------|----------------|-------------------|---------------------------|
| ⚛️ React (w/ Next.js 14) | 🛠️ API Routes / Server Actions | 🎨 Tailwind CSS + ShadCN | 💳 Stripe Payments |
| ⛑️ TypeScript          | 🗃️ PostgreSQL + Prisma | ✨ Framer Motion          | 🔍 Meilisearch/Algolia |
| 🔐 NextAuth / Clerk     | 🧠 Zod (Validation) | 🧱 Radix + Lucide Icons  | 🚀 Vercel (CI/CD)       |

---

## ✨ Features

- 🔍 **Search & filter books** with instant results
- 📦 **Product pages** with title, author, price, and stock
- 🛒 **Shopping cart** (local or persisted)
- 💳 **Checkout integration** with Stripe
- 🔐 **User authentication** (with Clerk or NextAuth)
- 🛠️ **Admin dashboard** for managing inventory
- 📱 **Responsive design** for mobile & tablet
- 🎨 **Polished UI** with dark mode and animations

## 🛠️ Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/book-shop.git](https://github.com/cariinacsantos/bookology.git
cd bookology

# Install dependencies
npm install

# Setup environment variables
cp (SOON)

# Setup the database
npx prisma migrate dev

# Run locally
npm run dev
